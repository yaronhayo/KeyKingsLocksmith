## 🚀 Deployment

### Quick Deploy

Upload all files to your web hosting root directory.

### Vercel (Recommended)
```bash
npm i -g vercel
vercel --prod
```

### Netlify
Drag and drop this folder to https://app.netlify.com/drop

### Traditional Hosting
Upload all files via FTP to your public_html or www folder.
